# Flags do jogador

Essa extensão implícita aos pacotes referentes à ela.\
Ela determina quais são as flags do jogador.

## Enumerações

Todas as enumerações devem ser usadas a partir da variável **PlayerFlags**.\
Por exemplo: `PlayerFlags.PIF_SWAPSIDE`

### PIF_SWAPSIDE

Jogador com volante à esquerda.

### PIF_AUTOGEARS

Jogador com o câmbio automático.

### PIF_SHIFTER

Jogador usando H-Shifter.

### PIF_HELP_B

Jogador usando ajuda do freio.

### PIF_AXIS_CLUTCH

Jogador usando eixo de embreagem.

### PIF_INPITS

Jogador dentro da garagem.

### PIF_AUTOCLUTCH

Jogador usando auto-embreagem.

### PIF_MOUSE

Jogador usando direção baseada no mouse.

### PIF_KB_NO_HELP

Jogador usando direção baseada no teclado (sem ajuda).

### PIF_KB_STABILISED

Jogador usando direção baseada no teclado (com ajuda).

### PIF_CUSTOM_VIEW

Jogador usando uma visão personalizada.
