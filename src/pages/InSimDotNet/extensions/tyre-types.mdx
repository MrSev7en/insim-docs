# Tipos dos pneus

Essa extensão implícita aos pacotes referentes à ela.\
Ela determina quais são os tipos dos pneus.

## Variáveis

Todas as variáveis devem ser usadas a partir da variável **Tyres**.\
Por exemplo: `Tyres.FrontLeft`

### FrontLeft

Representa os [Compostos de pneus](/InSimDotNet/extensions/tyre-compounds) frontal esquerdo.

### FrontRight

Representa os [Compostos de pneus](/InSimDotNet/extensions/tyre-compounds) frontal direito.

### RearLeft

Representa os [Compostos de pneus](/InSimDotNet/extensions/tyre-compounds) traseiro esquerdo.

### RearRight

Representa os [Compostos de pneus](/InSimDotNet/extensions/tyre-compounds) traseiro direito.
