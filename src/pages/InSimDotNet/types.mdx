# Tipos de pacotes

Existem dois tipos de pacotes, os receptores, e enviadores.

## Receptores

Receptores são pacotes que recebem dados do jogo, eles não podem ser manipulados ou enviados à nenhum jogador.
Na maioria das vezes esse pacote contém dados específicos, como o comando que um usuário enviou, ou quando ele entrou no jogo, por exemplo.

Alguns exemplos desse pacote são:

- Receber comandos dos jogadores.
- Identificar quando um jogador clica em algum botão.
- Identificar quando um jogador sai da garagem.

## Enviadores

Enviadores são pacotes que mandam dados para manipulação do jogo, enviados ao servidor, ou à um jogador específico, dependendo do pacote.

Alguns exemplos desse pacote são:

- Criar botões na tela.
- Teletransportar um jogador.
- Manipular semáforos.
