# Tipos de jogador

Essa extensão implícita aos pacotes referentes à ela.\
Ela determina qual é o tipo de jogador.

## Enumerações

Todas as enumerações devem ser usadas a partir da variável **PlayerTypes**.\
Por exemplo: `PlayerTypes.PLT_FEMALE`

### PLT_FEMALE

Jogador é uma mulher.

### PLT_AI

Jogador é uma inteligência artificial.

### PLT_REMOTE

Jogador está conectado de forma remota, e não local.
