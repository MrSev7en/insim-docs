# Flags do setup

Essa extensão implícita aos pacotes referentes à ela.\
Ela determina quais são as flags do setup.

## Enumerações

Todas as enumerações devem ser usadas a partir da variável **SetupFlags**.\
Por exemplo: `SetupFlags.SETF_SYMM_WHEELS`

### SETF_SYMM_WHEELS

As rodas estão simétricas.

### SETF_TC_ENABLE

O controle de tração está ativado.

### SETF_ABS_ENABLE

O sistema de anti-bloqueio de freio está ativado.
